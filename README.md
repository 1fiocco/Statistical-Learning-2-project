# Statistical-Learning-2-project
This is the final project for the second part of Fundamentals of Statistical Learning course, helded by the teacher Luca Tardella in the academic year 2025 / 2026 during the spring semester.


In the file named bayesian_irt_with_aic_bic_dic_predrep.html you will find the main results of the project, with a detailed explanation of what the project is about, the main findings and why they are interesting.


In the file named no_total_deviance_trace you will find the actual code that I used. It is an r code. In order for it to run smootly you have to set up some values into the code, like the modality (more info in the code itself), and of course you have to download the dataset (I uploaded it here as well).

Jags is required for the project to run smoothly.
